# VolunteerFinder

A dashboard for discovering volunteering opportunities near you, powered by Google Search (via Serper.dev).

## Setup

1. Install dependencies: `npm install`
2. Copy `.env.local.example` to `.env.local` and add your Serper API key
   - Get a free key at https://serper.dev (2,500 queries/month free)
3. Run: `npm run dev`
4. Open http://localhost:3000

## Running Tests

```bash
npm test
```
